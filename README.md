# 2-D-Watertank-filling-model-
Nonlinear 2-D tank filling model solved using finite difference method and enhanced with machine learning for fast water level prediction.
This project presents a nonlinear 2-D tank filling model based on the Forced Damped Duffing Equation to study water surface oscillations during filling. The governing equation is solved using the Finite Difference Method in MATLAB to simulate time-dependent water level variation. Additionally, machine learning techniques are implemented in Python to predict water height efficiently without repeatedly solving the differential equation. The project demonstrates the integration of numerical methods and data-driven modeling for real-world engineering applications such as fluid storage systems and smart tank monitoring.
